# UNP_PROJECT_HEALTH_-CARE_DATA_ANALYSIS_USING_MySQL

I completed this project in collaboration with UNP,This project focuses on analyzing healthcare data using MySQL, a relational database management system. The main objective is to store, manage, and analyze patient and hospital-related data efficiently to support better decision-making.
In this system, different types of data such as patient details, doctor information, appointments, treatments, and billing records are organized into tables in a database. Using SQL queries, the data is retrieved and analyzed to identify useful insights.
The project helps in:
Tracking patient records and medical history
Managing doctor and appointment schedules
Analyzing treatment outcomes and hospital performance
Generating reports for decision-making
By using MySQL, large volumes of healthcare data can be handled easily, ensuring accuracy, security, and quick access to information. This improves hospital management and enhances the quality of healthcare services.
